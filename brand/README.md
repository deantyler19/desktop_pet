# Dot 브랜드 에셋

펫 앱 "Dot(도트)"의 심볼마크. **"PET" 워드마크** — Arial Rounded 기반의 둥근 글자에 펫 요소를 녹임: 글자 위 **반짝이는 눈**, 글자 아래 **작은 발**(걸어다니는 단어 느낌).

## 파일
- `dot-wordmark.svg` — 벡터 마스터. 글자가 패스로 변환돼 있어 폰트 없이도 어디서나 동일하게 렌더. 색은 `fill` 값만 바꾸면 됨(코랄 `#bf5e3b`, 눈 `#221b18`).
- `dot-wordmark-coral-{128..1024}.png` — 메인(코랄 글자 + 검정 반짝눈), 투명 배경.
- `dot-wordmark-black-{128..512}.png` — 단색 흑(밝은 배경용).
- `dot-wordmark-white-{128..512}.png` — 화이트(어두운 배경용).

## 사용처
- **트레이 아이콘**: 루트 `tray.png` = 코랄 워드마크를 정사각으로 패딩한 64px.
  - 참고: 워드마크는 가로로 길어 16px 같은 초소형에선 작게 보임. 아주 작은 자리는 컴팩트 마크가 더 또렷할 수 있음.
- **앱 아이콘**: 루트 `icon.ico` = 마스코트(둥근 캐릭터 카드).

## 색상
- 코랄(브랜드): `#bf5e3b`
- 잉크(눈/흑): `#221b18`
- 화이트: `#ffffff`
